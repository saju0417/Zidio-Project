Zidio Connect - Full Stack
Backend: Spring Boot (Java 17)
Frontend: React + Tailwind

Database: MySQL (configured to root/1234 on port 3306)

How to run backend:
1. Import backend as Maven project in Eclipse.
2. Update application.properties if needed (backend/src/main/resources/application.properties).
3. Run as Spring Boot App or: mvn spring-boot:run

How to run frontend:
cd frontend
npm install
npm start

Docker:
docker-compose up --build
